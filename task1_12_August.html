Task 2 Description

        
Create a library called DummySelect which should have the following behavior:

A new DummySelect should be created like so:

var dummySelect = new DummySelect([args]);

The DummySelect constructor takes:
1. an array of strings: string1, string2, string3… stringN
2. The currently selected element
3. The currently moused over element


It creates an HTML div that looks like the following:

Each line represents one string in the array passed in to the function. For example, the box above would be generated by calling: var dummySelect = new DummySelect([“one”, “two”, “three”], 0, 0)

Here’s another example:

var dummySelect = new DummySelect([“one”,”two”,”three”], 0, 1)


The DummySelect constructor should return a reference to the node with the HTML in it that can then be inserted into the page.

The DummySelect object should have method on it called “highlight”. It should be called like this:

var dummySelect = new DummySelect(“one”,”two”,”three”)
dummySelect.highlight(2);

When “highlight” is called, it should go to the div specified by the input - in this case the div with the value “three” - and put a CSS class on it to highlight it.

The DummySelect object should also have a method on it called “select”. It should be called like this:

dummySelect.select(1);

This method should add a check before the selected element and remove the check from the previously selected element.
